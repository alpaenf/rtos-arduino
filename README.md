# 🚀 RTOS Arduino - LED Multitasking

Implementasi sederhana **Real-Time Operating System (RTOS)** menggunakan FreeRTOS pada Arduino Uno.

---

## 📌 Deskripsi

Project ini mendemonstrasikan konsep multitasking pada mikrokontroler menggunakan RTOS dengan dua task:

* Task 1: LED merah berkedip setiap 1 detik
* Task 2: LED kuning berkedip setiap 0.5 detik

Kedua task berjalan secara bersamaan menggunakan scheduler FreeRTOS tanpa blocking.

---

## ⚙️ Teknologi

* Arduino Uno
* FreeRTOS
* Wokwi Simulator
